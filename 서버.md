# 서버 실행 명령어

개발 모드

```
cd Server
npm run start:dev
```

프로덕션 빌드/실행

```
cd Server
npm run build
npm run start:prod
```

디버그 모드(선택)

```
cd Server
npm run start:debug
```

프론트 서버 실행

```
cd _frontServer
npm install
npx expo start --ios --localhost -c
```

웹/안드로이드(선택)

```
cd _frontServer
npx expo start --web --localhost -c
npx expo start --android --localhost -c
```

Prisma 실행

```
cd Server
# Prisma Client 생성
npm run db:generate

# 현재 스키마를 DB에 반영(push)
npm run db:push

# 개발용 마이그레이션 생성/적용(dev)
npm run db:migrate

# Prisma Studio(웹 UI)
npm run db:studio

# 시드 데이터 입력(있는 경우)
npm run db:seed
```
