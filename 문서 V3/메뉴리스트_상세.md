# 메뉴 리스트 상세 (V3)

전시(專時) - 전문가 유휴시간 파트타임/프리랜서 매칭 플랫폼의 메뉴별 상세 구성입니다.

---

## 1. 메인 탭 (Bottom Tab)

### 1-1. 단기 의뢰 `ProfessionalShortTime`

> **목적**: 기업이 전문가에게 시간 패키지/단기 프로젝트를 의뢰하고, 전문가가 요청을 확인/응답하는 핵심 매칭 공간

#### `ProfessionalShortTimeList` — 단기 의뢰 목록

**화면 구성**
- 상단: 검색바 + 필터 버튼
- 필터 옵션:
  - 분야: IT/소프트웨어, 마케팅, 재무/회계, HR, 디자인, 전략/컨설팅 등 (20개 대분류 기준)
  - 예산 범위: ~50만원, 50~100만원, 100~200만원, 200만원~
  - 기간: 1주 이내, 1개월 이내, 3개월 이내, 협의
  - 근무 형태: 원격, 대면, 혼합
- 의뢰 카드 리스트:
  - 기업명/로고
  - 의뢰 제목
  - 필요 역할 태그 (예: CMO, CFO, PM)
  - 예산 범위
  - 희망 기간
  - 등록일/마감일
  - 지원자 수

**주요 기능**
- 무한 스크롤 페이지네이션
- 즐겨찾기(북마크) 추가
- 최신순/예산순/마감임박순 정렬

---

#### `ProfessionalShortTimeListDetail` — 단기 의뢰 상세

**화면 구성**
- 기업 정보 섹션:
  - 기업명, 로고, 업종, 규모(인원수)
  - 서비스/제품 소개
  - 웹사이트 링크
- 의뢰 상세 섹션:
  - 의뢰 제목
  - 필요 역할 (예: "주 10시간 마케팅 리드")
  - 상세 업무 내용
  - 필요 역량/경험
  - 희망 협업 형태 (주 X시간 / 월 Y시간 / 월 N회 미팅)
  - 예산 범위
  - 희망 시작일 / 기간
  - 선호 커뮤니케이션 방식 (슬랙, 화상미팅, 대면 등)
- 하단 버튼:
  - [제안서 보내기] — 전문가용
  - [채팅 문의하기]
  - [북마크]

**전문가 → 기업 제안서 작성 시 입력 항목**
- 제안 가격 (시간당/패키지)
- 제공 가능 시간대
- 접근 방식 / 예상 산출물
- 포트폴리오 링크 첨부
- 자기소개 메시지

---

#### `ProfessionalShortTimeInput` — 단기 의뢰 등록 (기업용)
**입력 필드**
1. 기본 정보
   - 의뢰 제목 (예: "스타트업 IR자료 작성 지원")
   - 필요 역할 선택 (드롭다운/태그)
   
2. 상세 요구사항
   - 업무 내용 (텍스트 에디터)
   - 필요 역량/경험 (태그 선택 + 자유 입력)
   - 현재 상황 설명 (선택)

3. 협업 조건
   - 희망 형태: 주 단위 / 월 단위 / 회차 단위
   - 예상 시간: 주 X시간 또는 월 Y시간
   - 희망 기간: 시작일 ~ 종료일
   - 예산 범위: 최소 ~ 최대

4. 선호 조건 (선택)
   - 선호 요일/시간대
   - 근무 형태 (원격/대면/혼합)
   - 커뮤니케이션 방식

5. 공개 설정
   - 전체 공개 / 초대 전문가만 공개

### 1-2. 전문가 `Experts`

> **목적**: 검증된 전문가 풀을 탐색하고, 적합한 전문가를 찾아 직접 요청을 보내는 공간

#### `ExpertProfessionalList` — 전문가 목록/검색

**화면 구성**
- 상단: 통합 검색바
- 필터 패널:
  - 전문 분야 (20개 대분류 기반)
    - IT/소프트웨어, 보안/인증, 데이터/AI, 인프라/클라우드
    - 비즈니스 전략, 프로젝트 관리, 경영 컨설팅
    - 디지털 마케팅, 브랜딩/PR, 세일즈
    - UI/UX, 그래픽 디자인, 영상/모션
    - 재무/회계, 법률, 세무/관세
    - HR/인사, 교육/트레이닝
    - 헬스케어, 공학/제조
  - 경력 연차: 3년 이하, 3~7년, 7~15년, 15년 이상
  - 시급 범위: ~5만원, 5~10만원, 10~20만원, 20만원~
  - 가용 시간대: 평일 오전, 평일 저녁, 주말
  - 평점: 4.5 이상, 4.0 이상

- 전문가 카드 리스트:
  - 프로필 사진
  - 이름/닉네임
  - 직무/역할 태그
  - 한 줄 소개
  - 경력 연차
  - 평균 평점 + 리뷰 수
  - 대표 시간 패키지 가격
  - 응답률/응답 시간

**정렬 옵션**
- 추천순 (AI 매칭 점수)
- 평점 높은순
- 응답 빠른순
- 가격 낮은순
- 경력 많은순

---

#### `ExpertProfessionalistDetail` — 전문가 상세

**화면 구성**

1. 프로필 헤더
   - 프로필 사진
   - 이름, 직무
   - 한 줄 소개
   - 평균 평점 ★4.8 (리뷰 23개)
   - 응답률 95% | 평균 응답 2시간

2. 전문 분야 섹션
   - 전문 분야 태그들
   - 보유 기술/스킬 태그

3. 경력 섹션
   - 회사명 | 역할 | 기간 | 주요 성과
   - (복수 등록 가능)

4. 포트폴리오 섹션
   - 링크 카드 (깃허브, 노션, 비핸스, 개인 사이트 등)
   - 대표 프로젝트 설명

5. **시간 패키지 섹션** ⭐ 핵심
   - 패키지 카드 리스트:
     - 패키지명 (예: "주 10시간 마케팅 리드")
     - 유형: 주간/월간/회차
     - 가격
     - 포함 업무 범위
     - 커뮤니케이션 방식
   - [패키지 선택하기] 버튼

6. 가용 시간 섹션
   - 주간 캘린더 형태로 가능 시간대 표시
   - (월) 19:00~22:00, (수) 19:00~22:00, (토) 10:00~18:00

7. 리뷰 섹션
   - 별점 분포 그래프
   - 리뷰 리스트:
     - 기업명 | 프로젝트 유형 | 평점 | 코멘트 | 날짜

**하단 버튼**
- [요청 보내기] — 기업용
- [채팅 문의하기]
- [북마크]

---

#### `ExpertProfessionalInput` — 전문가 프로필/역량 등록

**입력 섹션**

1. 기본 정보
   - 프로필 사진
   - 이름/닉네임
   - 직무/역할 (선택)
   - 한 줄 소개 (100자)
   - 상세 소개 (500자)

2. 전문 분야
   - 대분류 선택 (복수 가능, 최대 3개)
   - 세부 스킬 태그 추가

3. 경력 정보 (복수 등록)
   - 회사/조직명
   - 직책/역할
   - 근무 기간 (시작~종료)
   - 주요 업무/성과 설명

4. 포트폴리오
   - 링크 추가 (깃허브, 노션, 비핸스 등)
   - 대표 프로젝트 설명 (선택)

5. 가격 설정
   - 희망 시급 범위
   - 희망 월 패키지 가격 범위

6. 공개 설정
   - 프로필 공개 범위: 전체 / 매칭된 기업만 / 비공개
   - 본명 공개 여부
