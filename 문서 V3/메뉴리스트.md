# 메뉴 리스트 (V3.1 - Mockup 기반 수정)

다음은 mockup 폴더의 실제 HTML 파일들을 분석하여 수정한 전시(專時) 앱의 메뉴 구조입니다. 기존 IA와 실제 구현된 화면의 차이를 반영했습니다.

## 메인 탭 (main Tab)
- **홈**: `Home`
  - 메인 대시보드 (홈.html 기반)
  - 통계 카드 및 긴급/일반 채용 목록
  - 배너 슬라이더 및 추천 콘텐츠

- **단기 의뢰**: `ShortTermRequests`
  - `ShortTermRequestsList` — 단기 의뢰 목록 (단기 의뢰 목록.html 기반)
  - `ShortTermRequestsDetail` — 단기 의뢰 상세 (단기 의뢰 상세.html 기반)
  - `ShortTermRequestsApplicants` — 지원자 목록 보기 (단기_의뢰_상세_지원자보기.html 기반)
  - `ShortTermRequestsApplicantsManagement` — 지원자 심사/판단 (단기의뢰상세_지원자판단.html 기반)
  - `ShortTermRequestsInput` — 단기 의뢰 등록 (단기 의뢰 상세 등록화면.html 기반)

- **전문가**: `Experts`
  - `ExpertsList` — 전문가 프로필 목록 (전문가 프로필 목록.html 기반)
  - `ExpertsDetail` — 전문가 프로필 상세 (전문가 프로필 상세.html 기반)
  - `ExpertsRegistration` — 전문가 프로필 등록 (전문가 프로필 등록.html 기반)

- **교육**: `Education`
  - `EducationList` — 교육 과정 목록 (교육 등록 목록.html 기반)
  - `EducationDetail` — 교육 상세 (교육 상세.html 기반)
  - `EducationRegistration` — 교육 과정 등록 (교육 등록.html 기반)
  - `EducationStudents` — 교육 수강생 목록 (교육 수강생 목록.html 기반)

- **채용**: `Recruitment`
  - `RecruitmentList` — 채용 공고 목록 (01채용 공고 지원.html 기반)
  - `RecruitmentDetail` — 채용 공고 상세 (01채용 공고 지원뷰어.html 기반)
  - `RecruitmentApplicants` — 지원자 관리 (채용 지원자 관리.html 기반)
