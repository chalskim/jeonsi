# 메뉴 리스트 (V1)

다음은 전시(專時) IA(와이어프레임) 기반으로 정리한 앱의 메뉴 구조입니다. 화면 이름은 실제 라우트명 기준으로 표기하며, 와이어프레임 상 계획된 항목은 현재 구현과 차이가 있을 수 있습니다.

## 메인 탭 (main Tab)
- 단기 의뢰: `ShortTermRequests`( 일별로 단기의뢰 )
  - `ShortTermRequestsList` — 단기 의뢰 목록
  - `ShortTermRequestsDetail` — 단기 의뢰 상세
  - `ShortTermRequestsInput` — 단기 의뢰 등록
  - `ShortTermRequestsApplicants` — 단기 의뢰 지원자 보기
  - `ShortTermRequestsApplicantsManagement` — 단기 의뢰 지원자 관리

- 전문가: `Experts`
  - `ExpertsList` — 전문가 목록/검색
  - `ExpertsDetail` — 전문가 상세
  - `ExpertsRegistration` — 전문가 프로필/역량 등록

- 교육: `Education`
  - `EducationsList` — 교육 과정 목록
  - `EducationDetail` — 교육 상세(기관/일정)
  - `EducationRegistration` — 교육 과정 등록
  - `EducationStudents` — 교육 수강생 관리

- 구인: `Recruitment` - ( 정규직/파트타임/계약직/인턴 등 )
  - `RecruitmentList` — 구인 공고 리스트
  - `RecruitmentListDetail` — 구인 공고 상세
  - `RecruitmentInput` — 구인 공고 등록/수정

## 사이드 메뉴 (공통)
- 계정/프로필
  - `RegistrationPersonal` — 개인 정보 등록
  - `PersonalRegistrationEdit` — 개인 정보 수정
  - `RegistrationCompany` — 기업 정보 등록
  - `CompanyRegistrationEdit` — 기업 정보 수정

- 업무/콘텐츠
  - `ShortTermRequestsList` — 단기의뢰 등록 및 목록
  - `ExpertsList` — 전문가 등록 및 목록
  - `EducationsList` — 교육 과정 등록 및 목록
  - `RecruitmentList` — 구인 공고 등록 및 목록

  - `ShoppingCartCorp` — 기업 장바구니

  - `MyPageCorporate` — 내정보(기업)


## 관리자 메뉴
- `AdminManagement` — 사용자/콘텐츠 관리
- `QA_Answers` — Q&A 답변 관리
- `AnnounceManagement` — 공고 관리
- `NoticeNewsManagement` — 공지 관리
- `BannerManagement` — 베너 광고 관리

## 기타
- `NoticeNewsList` — 공지 사항
- `AlarmManagement` — 알림 관리
- `Login` / `Signup` — 로그인/회원가입
- `SuccessCases` — 성공사례
