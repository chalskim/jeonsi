# 메뉴 리스트 (V3)

다음은 전시(專時) IA(와이어프레임) 기반으로 정리한 앱의 메뉴 구조입니다. 화면 이름은 실제 라우트명 기준으로 표기하며, 와이어프레임 상 계획된 항목은 현재 구현과 차이가 있을 수 있습니다.

## 메인 탭 (main Tab)
- 단기 의뢰: `ProfessionalShortTime`( 일별로 단기의뢰 )
  - `ProfessionalShortTimeList` — 단기 의뢰 목록
  - `ProfessionalShortTimeListDetail` — 단기 의뢰 상세
  - `ProfessionalShortTimeInput` — 단기 의뢰 등록

- 전문가: `Experts`
  - `ExpertProfessionalList` — 전문가 목록/검색
  - `ExpertProfessionalistDetail` — 전문가 상세
  - `ExpertProfessionalInput` — 전문가 프로필/역량 등록

- 교육: `Education`
  - `EducationList` — 교육 과정 목록
  - `EducationListDetail` — 교육 상세(기관/일정)
  - `ExpertProfessionalInput` — 교육 과정 등록
  - `EduListDetail` — 교육 자료/수강 관리

- 구인: `Recruitment` - ( 정규직/파트타임/계약직/인턴 등 )
  - `RecruitmentList` — 구인 공고 리스트
  - `RecruitmentListDetail` — 구인 공고 상세
  - `RecruitmentInput` — 구인 공고 등록/수정

## 사이드 메뉴 (공통)
- 계정/프로필
  - `RegistrationPersonal` — 개인 정보 등록
  - `PersonalRegistrationEdit` — 개인 정보 수정
  - `RegistrationCompany` — 기업 정보 등록
  - `CompanyRegistrationEdit` — 기업 정보 수정

- 업무/콘텐츠
  - `ExpertRecruitmentList` — 단기의뢰 등록 및 목록
  - `ExpertProfessionalList` — 전문가 등록 및 목록
  - `EducationList` — 교육 과정 등록 및 목록
  - `RecruitmentList` — 구인 공고 등록 및 목록

- 일정/즐겨찾기
  - `ScheduleManager` — 일정관리
  - `BookmarkPersonal` — 북마크(개인)
  - `BookmarkCorporate` — 북마크(기업)

- 결제/장바구니
  - `PaymentManagementCorporate` — 기업 결제관리
  - `PaymentManagementPersonal` — 개인 결제관리
  - `ShoppingCartCorp` — 기업 장바구니
  - `ShoppingCartPerson` — 개인 장바구니

- 설정/도움말
  - `Settings` — 설정
  - (도움말) — 안내 페이지

- 마이페이지
  - `MyPagePersonal` — 내정보(개인)
  - `MyPageCorperation` — 내정보(기업)


## 관리자 메뉴
- `AdminManagement` — 사용자/콘텐츠 관리
- `QA_Answers` — Q&A 답변 관리
- `AnnounceManagement` — 공고 관리
- `NoticeNewsManagement` — 공지 관리
- `BannerManagement` — 베너 광고 관리

## 기타
- `NoticeNewsList` — 공지 사항
- `AlarmManagement` — 알림 관리
- `Login` / `Signup` — 로그인/회원가입
